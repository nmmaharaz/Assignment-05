What is the Document Object Model (DOM)?

The Document Object Model(DOM) is an application programming interface (API) for HTML and XML documents. The DOM represents a document with a logical tree. Each branch of the tree ends in a node, and each node contains objects. DOM methods allow programmatic access to the tree with one can change the structure or style or content of a document.

How do you select an element from the DOM?

JavaScript provides some of methods to select DOM elements.

getElementById:

The getElementById method allows select an element by Id. This method returns the element object if it finds an element with the specified id, otherwise it returns null.

getElementByClassName:

The getElementByClassName method allows select an element by Class Name. It returns a collect of all elements in the document with the specified class name.

getElementByTagName:

The getElementByTagName method allows select an element by Tag Name. It returns a collect of all elements in the document with the specified tag name.

querySelector:

The querySelector method allows select the first element that matches a specified CSS selector in the document.

querySelectorAll:

The querySelectorAll method allows select all element that matches a specified CSS selector in the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation it is very usefulness. Event delegation allows us to attach a single event listener to parent element that will fire for all descendants matching a selector, whether those descendants exist now or are added in the future.
